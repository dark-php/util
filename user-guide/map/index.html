A Map is a PHP array wrapped with common functions, that requires a string index. These functions provide more secure access to an array object by enforcing index types. For string indexes see Collection.

Extending Map

To use Map you must create your own class that extends it. Actions are then applied directly to your created class, for example:

class MyMap extends Map {
}

Map contains a constructor that takes an array and sets it to $items but this can be overriden to give you the freedom to set the items however you want.

Create

$items = array('Item 1', 'Item 2', 'Item 3');

// Create map from array
$map = new MyMap($items);

Get

Returns the item at given index

$item = $map->get(0);

Length

Returns the number of items in map

$length = $map->length();

Add

Adds given item to map

$map->add('Item 4');

Delete

Deletes item at given index

$map->delete(3);

Contains

Returns true if item is in map

$inMap = $map->contains('Item 1')

Keys

Returns an array of keys in map

$keys = $map->keys();

Key Exists

Returns true if given index exists

$keyExists = $map->keyExists(0);
