A Collection is a PHP array wrapped with common functions, that requires an integer index. These functions provide more secure access to an array object by enforcing index types. For string indexes see Map.

Extending Collection

To use Collection you must create your own class that extends it. Actions are then applied directly to your created class, for example:

class MyCollection extends Collection {
}

Collection contains a constructor that takes an array and sets it to $items but this can be overriden to give you the freedom to set the items however you want.

Create

$items = array('Item 1', 'Item 2', 'Item 3');

// Create collection from array
$collection = new MyCollection($items);

Get

Returns the item at given index

$item = $collection->get(0);

Length

Returns the number of items in collection

$length = $collection->length();

Add

Adds given item to collection

$collection->add('Item 4');

Delete

Deletes item at given index

$collection->delete(3);

Contains

Returns true if item is in collection

$inCollection = $collection->contains('Item 1')

Keys

Returns an array of keys in collection

$keys = $collection->keys();

Key Exists

Returns true if given index exists

$keyExists = $collection->keyExists(0);
